# 🏨 نظام إدارة الحجوزات المتكامل | Booking System Professional

هذا المشروع هو عبارة عن منصة ويب متطورة لإدارة وعرض الحجوزات، تم بناؤه باستخدام إطار العمل **Laravel** وقاعدة بيانات **MySQL**. يوفر النظام تجربة مستخدم سلسة لإضافة وتنظيم المواعيد مع لوحة تحكم إدارية شاملة.

## 🌟 المميزات الرئيسية (Core Features)
* **نظام حجوزات آلي:** يتيح للمستخدمين حجز المواعيد وتخزينها في قاعدة البيانات فوراً.
* **لوحة تحكم للمدير (Admin Dashboard):** واجهة احترافية لمراقبة جميع الحجوزات والتحكم بها.
* **إدارة كاملة (CRUD):** إمكانية الإضافة، العرض، التعديل، والحذف بكل سهولة.
* **نظام حماية واستيقاق (Authentication):** تأمين الدخول للمسؤولين فقط لإدارة البيانات الحساسة.
* **تصميم عصري (Responsive Design):** الواجهة متوافقة تماماً مع شاشات الجوال والأجهزة اللوحية.

## 🛠 التقنيات المستخدمة (Technical Stack)
* **Backend:** PHP (Laravel Framework)
* **Frontend:** Blade Templates, Bootstrap, CSS3
* **Database:** MySQL
* **Development Environment:** Laragon & Composer

## 🔘 دليل وظائف الأزرار (UI Guide)
| الزر | الوظيفة البرمجية |
| :--- | :--- |
| **إضافة حجز** | يقوم بعملية `POST` لبيانات العميل إلى قاعدة البيانات. |
| **تعديل (Edit)** | يسمح بتحديث بيانات الحجز عبر معرف فريد (ID). |
| **حذف (Delete)** | يقوم بمسح السجل نهائياً من النظام. |
| **تأكيد الحجز** | يغير حالة الحجز في قاعدة البيانات ليصبح "مؤكداً". |

## 🚀 كيفية التشغيل (Quick Start)
1. قم بتحميل المستودع (Clone) على جهازك.
2. أنشئ قاعدة بيانات باسم `booking_system_db` في Laragon.
3. نفذ أمر الهجرة لإنشاء الجداول: `php artisan migrate`.
4. ابدأ العمل: `php artisan serve`.
