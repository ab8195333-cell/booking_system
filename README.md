# 🏨 نظام إدارة الحجوزات المتكامل (Booking Management System)

نظام ويب متطور مبني بإطار العمل **Laravel**، يهدف إلى تسهيل عملية الحجز والدفع الإلكتروني مع نظام إشعارات فوري ولوحة تحكم ذكية.

## 🚀 الإنجازات التقنية في المشروع

تم تطوير هذا المشروع ليشمل الميزات الاحترافية التالية:

### 1. نظام إدارة الصلاحيات الذكي (Multi-Role System)
* **واجهة الأدمن:** تفتح تلقائياً عند تسجيل الدخول بالبريد المعتمد (`ab8195333@gmail.com`) وتوفر تحكماً كاملاً (عرض، تعديل، حذف) لجميع الحجوزات.
* **واجهة المستخدم:** توفر تجربة مستخدم أنيقة تعتمد على نظام البطاقات (Cards) لعرض الحجوزات الشخصية ومتابعة حالة الدفع.

### 2. الربط مع بوابة الدفع العالمية (Stripe Integration)
* دمج نظام **Stripe** لمعالجة المدفوعات بأمان، مع تحديث تلقائي لحالة الحجز في قاعدة البيانات بمجرد نجاح عملية الدفع.

### 3. نظام الإشعارات الفوري (Automated Notifications)
* **إرسال عبر Gmail:** تم ربط المشروع بنجاح مع خوادم Google SMTP باستخدام "App Passwords".
* **إشعارات العميل:** وصول رسالة تأكيد فورية للعميل عند إنشاء حجز جديد.
* **إشعارات الإدارة:** تنبيه فوري لبريد الإدارة عند وجود أي نشاط جديد في النظام.

### 4. العمليات الخلفية وقواعد البيانات (Backend & DB)
* بناء بنية بيانات قوية باستخدام **MySQL** مع علاقات (Relationships) بين المستخدمين والحجوزات.
* تنفيذ عمليات **CRUD** كاملة بطريقة احترافية ومؤمنة (Secure Controllers).

## 🛠 التقنيات المستخدمة (Tech Stack)
* **إطار العمل:** Laravel (PHP)
* **قاعدة البيانات:** MySQL / MariaDB
* **تنسيق الواجهات:** Bootstrap 5 & Google Fonts
* **بوابة الدفع:** Stripe API
* **خادم البريد:** Gmail SMTP

## 🔧 طريقة التشغيل (Setup)
1. قم بتحميل المشروع (Clone) وتثبيت المكتبات عبر `composer install`.
2. قم بتهيئة ملف `.env` ببيانات قاعدة البيانات و Stripe و App Password الخاص بـ Gmail.
3. قم بتشغيل التهجير: `php artisan migrate`.
4. ابدأ السيرفر: `php artisan serve`.
