# 🏨 نظام إدارة الحجوزات المتكامل - Integrated Booking Management System

نظام ويب متطور مبني باستخدام إطار العمل **Laravel** لإدارة عمليات الحجز والدفع الإلكتروني بشكل مؤتمت وآمن.

---

## 🚀 الميزات الرئيسية (Key Features)

* **💳 بوابة دفع Stripe:** دمج كامل لبوابة الدفع العالمية Stripe لمعالجة المعاملات المالية بأمان.
* **🔐 نظام المصادقة (Authentication):** تأمين كامل للوحة التحكم مع إمكانية تسجيل الدخول والخروج.
* **📊 إدارة الحجوزات (CRUD):** إمكانية إضافة، عرض، وتعديل الحجوزات بسهولة.
* **✅ تحديث الحالة تلقائياً:** يقوم النظام بتغيير حالة الحجز إلى "مكتمل" فور نجاح عملية الدفع عبر الـ Webhooks/Callbacks.
* **📱 واجهة مستخدم متجاوبة:** تصميم عصري باستخدام Bootstrap يدعم جميع الشاشات واللغة العربية (RTL).

---

## 🛠 التقنيات المستخدمة (Tech Stack)

* **Backend:** PHP 8.x, Laravel 10/11.
* **Database:** MySQL.
* **Frontend:** Blade Engine, Bootstrap 5.
* **Payment Gateway:** Stripe API.
* **Tools:** VS Code, Git/GitHub, Laragon.
